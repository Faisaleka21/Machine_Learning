# Machine Learning

Repository ini berisi kumpulan source code dan latihan algoritma Machine Learning menggunakan Python dan Google Colab.

## Cara Membuka Source Code di Google Colab

Jika file Notebook (`.ipynb`) tidak dapat dibuka langsung melalui GitHub, gunakan Google Colab dengan menambahkan URL berikut:

```text
https://colab.research.google.com/github/
```

di depan path file notebook GitHub yang ingin dibuka.

### Format Umum

```text
https://colab.research.google.com/github/USERNAME/NAMA_REPOSITORY/blob/main/NAMA_FILE.ipynb
```

### Contoh

Source GitHub:

```text
https://github.com/Faisaleka21/Machine_Learning/blob/main/jawaban_no4_ML-REGRESI-LINEAR-BERGANDA.ipynb
```

Buka melalui Google Colab:

```text
https://colab.research.google.com/github/Faisaleka21/Machine_Learning/blob/main/jawaban_no4_ML-REGRESI-LINEAR-BERGANDA.ipynb
```

## Langkah-Langkah

1. Buka repository GitHub.
2. Pilih file notebook (`.ipynb`) yang ingin dijalankan.
3. Salin URL file tersebut.
4. Tambahkan:

```text
https://colab.research.google.com/github/
```

5. Hapus bagian:

```text
https://github.com/
```

6. Tempel URL yang telah disesuaikan ke browser.
7. Notebook akan terbuka otomatis di Google Colab.
8. Klik **Run All** untuk menjalankan seluruh program.

## Alternatif

Anda juga dapat membuka Google Colab secara langsung:

https://colab.research.google.com

Kemudian pilih:

**File → Open Notebook → GitHub**

Lalu masukkan URL repository GitHub yang diinginkan.

## Catatan

- Pastikan repository bersifat **Public** agar dapat diakses melalui Google Colab.
- Pastikan file yang dibuka memiliki ekstensi `.ipynb`.
- Jika dataset digunakan dalam notebook, unggah dataset terlebih dahulu sebelum menjalankan program.
